# Contributing to fakelink

Thank you for your interest in contributing to fakelink!

## Development Setup

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start development with hot reload
4. Run `npm run build` to create a production build

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Commit your changes (`git commit -m 'Add my feature'`)
4. Push to the branch (`git push origin feature/my-feature`)
5. Open a Pull Request

## Reporting Issues

If you find a bug or have a feature request, please open an issue on GitHub with a clear description and steps to reproduce.
